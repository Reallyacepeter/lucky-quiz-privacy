속마음 리더 개인정보처리방침

최종 업데이트: 2026년 4월 20일

속마음 리더(이하 "앱")는 이용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 방침은 앱이 어떤 정보를 수집·이용·보관·공유하는지 설명합니다.

1. 수집하는 정보

앱은 이용자를 식별할 수 있는 정보를 수집하거나 저장하지 않습니다. 다음 정보만 이용자의 기기 내에서 처리됩니다.

  • 생년월일, 성별: 운세 콘텐츠 생성을 위해 사용됩니다. 이 정보는 이용자의 기기에만 저장되며 서버로 전송될 때 익명으로 처리됩니다.
  • 앱 캐시(SharedPreferences): 동일한 날 동일한 입력의 중복 API 호출 방지를 위해 생성된 운세 텍스트를 로컬에 저장합니다. 개인 식별 정보는 포함되지 않습니다.

2. 외부 API 이용

  • Google Gemini API: 입력된 생년월일·성별·날짜 정보를 운세 텍스트 생성을 위해 Google의 생성형 AI에 전달합니다. Google의 데이터 처리 정책은 Gemini API 약관을 따릅니다.
  • Google AdMob: 광고 제공을 위해 광고 ID 등 기술적 정보가 수집될 수 있습니다. 자세한 내용은 Google 광고 정책을 참조하세요.

3. 제3자 제공 및 보관

앱은 자체적으로 어떠한 개인정보도 서버에 보관하지 않으며, 마케팅·분석 목적으로 제3자에게 제공하지 않습니다. 본 앱에서 제공하는 운세 결과는 엔터테인먼트 목적이며, 상담·의료·법률·금융 등의 전문 조언으로 사용될 수 없습니다.

4. 아동 정보

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 정보를 의도적으로 수집하지 않습니다.

5. 권한

  • 인터넷: 운세 생성(Gemini API) 및 광고 표시(AdMob)를 위해 사용됩니다.

6. 이용자의 권리

이용자는 언제든지 앱을 삭제하여 로컬에 저장된 모든 정보를 삭제할 수 있습니다. 앱 설정 > 저장공간 > 데이터 지우기로도 캐시를 초기화할 수 있습니다.

7. 정책 변경

본 방침은 법령 개정, 앱 기능 변경 등에 따라 수정될 수 있습니다. 중요한 변경 시 앱 내 공지 또는 업데이트 노트로 안내합니다.
